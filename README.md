# Legacy Excel XLS Unlocker / DPB Modifier

A Python utility to unlock XLS files.
Reference: https://www.excelhelp.com/how-to-remove-excel-passwords-a-complete-guide-to-all-six-types/

**Purpose:**  
- Locate `DPB` records in legacy XLS files  
- Replace them with `DPx` for authorized testing/education  

**⚠️ Warning:**  
Only use on files you own or are authorized to modify.
